# Whitelisted-proxy

To run it
- npm i
- npm run dev

to build it
- npm run build
- > build dist in project folder

to serve it
- npm run serve

to fill whitelist:
1. go to file "whitelist.json"
2. fill it with allowed tokens like array
Example:
[
  "f4906093727730b10d349282cb0f0bc1",
  "b4916093727734b10d349282cb0f1bw1"
]
